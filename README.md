## Installation
```
$ git clone https://github.com/itayod/galacoral.git
$ cd galacoral
$ npm install
```

## Development
```
$ npm start
```
Runs the project in development mode with hot-reloading of `public` folder. 
Open your browser at [http://localhost:8080](http://localhost:8080).
